# DarkSounls

这是一个关于傅老师的黑魂复刻Unity游戏项目，并根据个人习惯进行了项目调整

## 目录结构

```
Assets
├── Animations          // 动画文件夹
├── Audio               // 音频文件夹
├── Editor              // 自定义编辑器脚本文件夹
├── Fonts               // 字体文件夹
├── Materials           // 材质文件夹
├── PhysicsMaterials    // 物理材质文件夹
├── Models              // 3D模型文件夹
├── Prefabs             // 预制件文件夹
├── Scenes              // 场景文件夹
├── Scripts             // 脚本文件夹
│   ├── Controllers     // 控制器类文件夹
│   ├── Managers        // 管理器类文件夹
│   ├── UI              // UI相关脚本文件夹
│   ├── Gameplay        // 游戏逻辑脚本文件夹
│   └── Utils           // 工具类文件夹
├── Shaders             // 着色器文件夹
├── Textures            // 纹理文件夹
└── UI                  // UI相关资源文件夹
    ├── Sprites         // 精灵文件夹
    └── UIElements      // UI元素文件夹
```
